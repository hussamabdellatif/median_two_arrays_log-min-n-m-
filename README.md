# median_two_arrays_log-min-n-m-
Find the median of the two sorted arrays with a time complexity of O(log min(m+n)).

Problem was Given By Leetcode... 

Example 1:
//
// nums1 = [1, 3]
// nums2 = [2]
//
// The median is 2.0
// Example 2:
//
// nums1 = [1, 2]
// nums2 = [3, 4]
//
// The median is (2 + 3)/2 = 2.5
